Agent 02 · Make.com · Free

Finnish Vocabulary
from the News

Every weekday at 17:00, your agent reads Yle Selkouutiset, asks Claude to pick the most useful Finnish words, and sends a vocabulary list to your Telegram.

Free to run ~30 min setup Finnish learners
// Agent workflow Screenshot from Make.com
🧰

What you'll need

Make.com account (free, 1000 ops/month) · Anthropic API key ( minimum) · Telegram account + bot token from @BotFather

Step-by-step guide
01

Create a new scenario in Make

Go to make.com, sign up (free, no credit card needed), and click "Create a new scenario".

02

RSS Feed — Watch Yle Selkouutiset

Click + on the canvas → search for RSS Feed → select "Watch RSS Feed Items".

  • URL: https://yle.fi/rss/selkouutiset
  • Maximum number of returned items: 1

When asked "Choose where to start" → select "From now on" → Save.

💡

About the RSS feed

Yle provides an official RSS feed for Selkouutiset — no scraping needed. The feed updates every weekday after 16:35 when the broadcast airs. On weekends there is no new episode, so the agent simply won't find anything new to process.

03

HTTP GET — Fetch the full article

Click + → search HTTP → "Make a request".

  • URL: click the field → from the left panel select RSS 1 → URL (this pulls the article link dynamically)
  • Method: GET
  • Parse response: Yes

This fetches the full text of today's Selkouutiset article — much more content than the RSS description alone.

04

HTTP POST — Send to Claude API

Click + → HTTP → "Make a request". Fill in:

  • URL: https://api.anthropic.com/v1/messages
  • Method: POST
  • Authentication type: No authentication

Add 3 headers:

Headers
x-api-key          →  YOUR_ANTHROPIC_API_KEY
anthropic-version  →  2023-06-01
content-type       →  application/json

Body content type: application/json → Body input method: Data structure → click "Create a data structure" → Generate from this JSON:

JSON
{"model": "claude-haiku-4-5-20251001", "max_tokens": 1024, "messages": [{"role": "user", "content": "text"}]}

After generating the structure, fill in the fields:

  • Model: claude-haiku-4-5-20251001
  • Max tokens: 1024
  • Messages → Add item → Role: user
  • Content: type your prompt text + select HTTP 7 → Data → Body from the left panel

See the Prompt Variants section below for ready-to-use prompt options.

🔑

How to get your Anthropic API key

Go to console.anthropic.com → sign up → top up with minimum → go to API Keys → click "+ Create key" → copy it immediately (shown only once).

⚠️

Keep your keys safe

Never share your API keys publicly or upload them to GitHub. Store them in a password manager or a private notes app on your device. If a key is accidentally exposed, delete it immediately and create a new one.

⚠️

Use Data structure, not JSON string

The article HTML contains line breaks and special characters that break raw JSON strings. Using "Data structure" lets Make escape them automatically.

05

Telegram Bot — Send the vocabulary list

First, create your bot:

  • Open Telegram → find @BotFather (blue checkmark) → send /newbot
  • Give it a name
  • Copy the token (looks like 7123456789:AAF...)
  • Important: send /start to your new bot — otherwise it can't message you
  • Get your Chat ID: find @userinfobot → send /start → copy your numeric ID

In Make: click + → search Telegram Bot → "Send a Text Message or a Reply".

  • Connection: create new → paste your bot token
  • Chat ID: your numeric ID
  • Text: type manually {{8.data.content[].text}} (replace 8 with your Claude HTTP module number)
👥

Want to send to multiple people?

Add a second Telegram Bot module after the first one with the same bot token but a different Chat ID. Each person must send /start to the bot first.

06

Set the schedule and activate

  • Click the clock icon on the RSS module → Schedule settings
  • Run scenario: Daily
  • Time: 17.00
  • Time zone: Europe/Helsinki (set automatically)

Click Save, then click "Activate scenario". Done — your agent will run automatically every weekday!
