Simple AI Agents for Beginners

Build your first AI Agent in 30 min

A step-by-step guide to creating an automated AI news digest that runs every morning — no coding required.

No code required Free to start ~30 min setup

What you'll need

n8n.io account (free trial) · NewsAPI key (free) · Anthropic API key ($5 minimum) · Telegram account

Step-by-step guide
01

Schedule Trigger

In n8n, create a new workflow. Click the + button on the canvas and search for Schedule Trigger.

  • Trigger Interval: Days
  • Days Between Triggers: 1
  • Trigger at Hour: 8am
  • Trigger at Minute: 0

This will run your agent every morning at 8:00 automatically.

02

HTTP Request — Fetch News

Click + after the trigger → search for HTTP Request. Set Method to GET and paste this URL (replace YOUR_API_KEY with your key from newsapi.org):

URL 
https://newsapi.org/v2/everything
  ?q=artificial+intelligence
  &language=en
  &sortBy=publishedAt
  &pageSize=5
  &apiKey=YOUR_API_KEY

Click Execute step — you should see articles with status: ok in the Output.

💡

Filter out irrelevant news

NewsAPI sometimes returns loosely related articles. To improve precision, use: q=%22artificial+intelligence%22+OR+%22machine+learning%22

03

Anthropic — Process with Claude

Click + → search for Anthropic → select Message a Model.

  • Click Set up credential → paste your API key from console.anthropic.com
  • Model: claude-haiku-4-5-20251001 (cheapest, ~$0.001 per digest)
  • Paste your prompt in the Message field — see the Prompt Variants section below for ready-to-use options
💰

Cost estimate

One daily digest costs approximately $0.001. A $5 credit will last you over 13 years of daily use.

04

Telegram Bot — Send the Digest

First, create your bot and get your Chat ID:

  • Open Telegram → find @BotFather (blue checkmark) → send /newbot
  • Follow instructions → copy the token (looks like 7123456789:AAF...)
  • Find @userinfobot → send /start → copy your numeric ID
  • Important: send /start to your new bot — otherwise it can't message you

Back in n8n: click + → search Telegram → Send a Text Message.

  • Create new credential → paste your bot token
  • Chat ID: your numeric ID from @userinfobot
  • Text field:
n8n Expression 
{{ $('Message a model').item.json.content[0].text }}
05

Activate the Agent

Click Execute workflow to test. If you receive a Telegram message — everything works!

  • Click the Publish button (top right)
  • The green dot Published = agent is active
  • From now on it runs automatically every morning at 8:00
⚠️

Gemini API note for EU users

Google Gemini free API is not available in the EU due to regulations. Use Claude (Anthropic) instead — it works perfectly from Finland and other EU countries.

Prompt variants

Choose the prompt style that fits your needs. Paste it into the Message field of the Anthropic node.

Concise bulletin concise
Here are today's fresh AI news articles: {{ $('HTTP Request').item.json.articles.map(a => a.title + ': ' + a.description).join('\n\n') }} Write a 5-point digest in English. Each point: 2-3 sentences. Style: friendly and clear, for a general audience. Ignore any articles not related to artificial intelligence.
Detailed briefing with context detailed
Here are today's AI news articles: {{ $('HTTP Request').item.json.articles.map(a => a.title + ': ' + a.description).join('\n\n') }} Create a morning AI briefing with: - A one-sentence headline summary - 5 key stories, each with: what happened, why it matters - One "big picture" takeaway at the end Tone: professional but conversational. Skip anything not AI-related.
Ultra-short for busy people minimal
AI news: {{ $('HTTP Request').item.json.articles.map(a => a.title + ': ' + a.description).join('\n\n') }} Give me 5 bullet points. Max 1 sentence each. Only AI-related news. Start each with an emoji.
Краткий дайджест кратко
Вот свежие новости про AI: {{ $('HTTP Request').item.json.articles.map(a => a.title + ': ' + a.description).join('\n\n') }} Напиши дайджест на русском языке: 5 коротких пунктов, каждый 2-3 предложения. Стиль — дружелюбный, для широкой аудитории. Если среди новостей есть статьи не связанные с искусственным интеллектом — игнорируй их.
Подробный разбор с контекстом подробно
Вот сегодняшние новости про AI: {{ $('HTTP Request').item.json.articles.map(a => a.title + ': ' + a.description).join('\n\n') }} Создай утренний AI-брифинг на русском: - Одно предложение — главная тема дня - 5 новостей: что произошло + почему это важно - Один общий вывод в конце Тон: профессиональный, но живой. Пропусти всё не связанное с AI.
Ультракороткий формат минимум
Новости AI: {{ $('HTTP Request').item.json.articles.map(a => a.title + ': ' + a.description).join('\n\n') }} Дай 5 пунктов. Максимум одно предложение каждый. Только новости про AI. Начни каждый пункт с эмодзи. Пиши на русском.
